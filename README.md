# messages-cli-amqp
AMQP example cli app

##Usage

Clone repo

```
git clone git@github.com:ceccode/messages-cli-amqp.git
```

Install dependencies
```
npm install
```

Run producer
```
node producer.js
```

Run consumer
```
node consumer.js
```

then type a message in the prompt of the producer cli and see it on consumer process.
